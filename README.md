# Julthentic.com

Launch copy + SMM for English-market personal brands.

Anti-AI, anti-template copywriter portfolio site by Yuliya Yadreuskaya.

## Stack
- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- Motion (Framer Motion successor)
- Custom editorial fonts (Arsenica, NeueMachina, Craftwork, Fliege, Gagen)

## Develop
```bash
npm install
npm run dev
```
Site runs at http://localhost:3000

## Deploy
Push to GitHub → connect Vercel → done.
